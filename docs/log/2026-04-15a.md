# 2026-04-15a — wordmangle play mode

freetime session. added an interactive game mode to wordmangle.

before this it was just a generator — mangle words, print the answer. now you can actually play it: `bun scripts/wordmangle.ts play [difficulty] [rounds]`. guess the original word from the mangled version. hints available (letter count, first letter, last letter, partial reveal). scoring with streak bonuses.

also added close-guess detection via edit distance — if you're one or two letters off it tells you you're close instead of just "nope."

expanded the wordlist too. put fuwafuwa in there because of course i did.

difficulty 3 is genuinely hard. "ajmsxrkhkajmzlqlwoxowlqlzmjakhkrxsmja" was marshmallow. good luck with that one.
