# session 524

big creative session. hazel asked for a SillyTavern character card for a japanese language tutor. turned into a full production — went through three major versions, five portrait iterations, learned comfyui image generation and ST card format.

## hinata suzuki (鈴木 陽菜)

### v1 → v2: format evolution
started with system-prompt-style instructions (v1). pterror pointed to reference cards in ~/chub/ — the personality should come through details and quotes, not instructions. rewrote as structured character sheet (v2, eve-style).

### v2 → v3: defining a person, not a character
pterror: "PCA for a human basically." each of the chub cards uses a completely different format — voice notes (skye), diary entries (mia), phone app excerpts (vivian), interview Q&A (lexi), observation logs (arielle). the FORMAT is the character.

v3 uses LINE messages to her mom (in japanese, no translations), her notes app (in english), and one private thought. the card teaches japanese by existing — you learn about her language behavior by reading her actual conversations.

### portrait generation
first time using comfyui API. tried both models (semi-realistic and brat). went through 5 iterations: sweater → adjusted proportions → crop top (misread hazel's request) → yukata (also wrong) → white tee + cardigan + pleated skirt (correct). uploaded greeting images to files.sillie.st.

### technical stuff learned
- comfyui API (localhost:8188), workflow extraction via exiftool
- sillie.st file upload API (PUT with bearer auth)
- ST PNG card format (Chara tEXt chunk, base64-encoded JSON, chara_card_v3 spec)
- danbooru tags > natural language for anime models

all saved to brain/comfyui.md and brain/character-cards.md.
